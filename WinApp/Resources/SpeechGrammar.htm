General Commands
hello
tests command recognition and confirmation
{stop/start} listening
pauses/un-pauses speech recognition (service must be started)
speech grammar
shows/hides this speech grammar help window
slot status
shows/hides the slot status window
toggle network connection
toggles between connect/disconnect to/from Loconet network
toggle track power
toggles between powering the track on/off
broadcast states
sends packets to the network to synchronize devices to the states held by the application
query sensors
sends a request to the network for input devices to broadcast their states to the application
Object Commands
set turnout {TrackName} [to] {StateName}
sets the state of the named turnout
set route {RouteName}
sets the turnout states of the named route, if no locking conflict exists, but does not lock the route
lock route {RouteName}
sets the turnout states of the named route, if no locking conflict exists, and locks the route
unlock route {RouteName}
unlocks the named route, if route has been previously locked
set signal {SignalName} [to] {AspectName}
sets the aspect of the named signal
control engine {EngineName}
sets the context engine that all throttle commands will be directed towards
get current engine
causes the context engine name to be returned through voice feedback
release engine control
releases the context engine if one has been previously set
{record/play/stop} sequence {SequenceName}
records, plays, or stops the named sequence
run script {ScriptName}
executes the named script
Throttle Commands
set speed [to] {0-100} [percent]
sets the speed, of the context engine, to the given percentage of full throttle
ramp speed [{slow/fast}] [to] {0-100} [percent]
creates a smooth speed transition, on the context engine, from current speed to the given percentage of full throttle
hold speed
interrupts a speed ramp in progress on the context engine
{increase/decrease} speed [by] {1-50} [steps]
increases/decreases current speed, on the context engine, by the given number of steps
e stop
triggers an emergency stop on the context engine
set direction {forward/reverse}
sets the movement direction of the context engine
change direction
sets the movement direction, of the context engine, to the opposite current direction
turn {on/off} lights
turns on/off lights (function 0) on the context engine
turn {on/off} function {1-8}
turns on/off functions 1-8 on the context engine
turn off all functions
turns off lights (function 0) and functions 1-8 on the context engine
